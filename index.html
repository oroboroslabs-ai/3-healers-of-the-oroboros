Quick Install

Get started with the 3 Healers in under 5 minutes using Ollama.

Prerequisites

  • ✓ Ollama installed
  • ✓ 8GB+ RAM recommended
  • ✓ Terminal/Command Line access

Installation

# Clone the repository
git clone https://github.com/oroboroslab/3-healers.git
cd 3-healers

# Build Hygeia (Holistic Health)
./build_hygeia.sh

# Build Aria (Neuroscience)
./build_aria.sh

# Build Rheya (Natural Medicine)
./build_rheya.sh

# Chat with any healer
ollama run hygeia
ollama run aria
ollama run rheya
